# ⚡ ELITE IA - Entrenamiento Físico Inteligente

![ELITE IA Hero Banner](./atleta_hero_barra.png)

> **Revoluciona tu entrenamiento con el poder de la Inteligencia Artificial.**

**ELITE IA** es una landing page moderna, inmersiva y altamente responsiva para una plataforma de entrenamiento físico potenciado por Inteligencia Artificial. Este proyecto se diseñó con un fuerte enfoque en proporcionar una experiencia de usuario (UX) premium y una interfaz de usuario (UI) de alto impacto. 

Fue desarrollado como parte del trabajo práctico para la **Diplomatura en Diseño UX/UI (Unidad 2: Introducción a la Inteligencia Artificial)**.

---

## ✨ Características Principales

- **Diseño Oscuro y Elegante (Dark Mode):** Interfaz inmersiva con acentos en color neón (#ccff00) para transmitir una estética *premium* y de alto rendimiento, similar a plataformas deportivas líderes.
- **Diseño Responsivo (Mobile-First):** Completamente adaptable a pantallas de escritorio, tablets y dispositivos móviles.
- **Micro-interacciones Fluidas:** Efectos hover modernos y transformaciones (`scale`) para botones y mockups.
- **Formulario de Inscripción Interactivo (Modal):** Ventana emergente integrada para captura rápida de leads sin tener que redireccionar al usuario a otra URL, reduciendo pasos y fricciones en la inscripción.
- **Tipografía y Legibilidad:** Uso de `Poppins` como fuente digital moderna y Google Material Symbols para la iconografía adaptativa.
- **Enlaces Redes Sociales Dinámicos:** Iconos SVG personalizados para WhatsApp e Instagram, incrustados directamente y configurados para abrir nuevas interfaces de chat.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica sólida y clara de los diferentes componentes de la landing.
- **Tailwind CSS (mediante CDN con configuración en-línea):** Framework de utilidades (`utility-first`) para construir el diseño y la paleta de colores sobre la marcha, incluyendo animaciones y *backdrop-filters*.
- **JavaScript (Vanilla):** Utilizado de manera ágil para la lógica de apertura y cierre del Formulario Modal de inscripción.
- **Google Fonts:** Tipografías (*Poppins*).
- **Google Material Symbols:** Iconos fluidos en la cabecera, botones, tarjetas e integraciones.

## 🚀 Cómo Visualizar el Proyecto

La página es estática, por lo que no es necesaria la instalación de pesados paquetes de `npm`.

1. Clona este repositorio en tu ordenador usando Git (o descarga el `.zip` del código fuente):
   ```bash
   git clone https://github.com/TU-USUARIO/elite-ia-entrenamiento.git
   ```
2. Asegúrate de tener una conexión a Internet activa. Esto es requerido para que la hoja de estilos de Tailwind y las fuentes que provienen de los enlaces o CDN puedan cargar correctamente las métricas en tu navegador.
3. Haz doble clic en el archivo llamado `entrenamiento.html` y ábrelo visualmente en tu navegador web de preferencia (Chrome, Safari, Firefox o Edge).
   - *Tip Pro: Si tienes la extensión "Live Server" en Visual Studio Code, haz click derecho sobre `entrenamiento.html` y presiona "Open কমপ্লেy Live Server" para ver los cambios en tiempo real.*
